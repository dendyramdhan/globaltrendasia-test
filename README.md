# Global Trend Asia Coding Test

## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/dendyramdhan/globaltrendasia-test.git`

### Setup Web Service

> install all library first

```shell
$ cd web-service
$ yarn
```

> now migrate and seed DB

```shell
$ yarn migrate
$ yarn seed
```

> run server

```shell
$ yarn start
```

### Setup Android App

> install all library first

```shell
$ cd GlobalTrendAsia
$ yarn
```

> run android app

```shell
$ yarn android
```
